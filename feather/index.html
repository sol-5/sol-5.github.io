Feather CSS

A Minimalistic, Responsive, CSS Library for building Intuitive UIs

Features

Lightweight

Feather is a tiny library consisting of ~500 Lines of Code. It provides all the basic components that are needed in building unique User Interfaces.

Responsive

Feather is Responsive by Default. Your websites will scale and adapt fluidly according to your user's device.

Minimal

Feather incorporates the Latest trends in UI Design and the result is websites with Clean and Modern look.

Customizable

Feather leverages the power of CSS Variables to provide complete customizability. These variables can be used to quickly build custom styles.
